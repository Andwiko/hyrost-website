# Hyrost Website

Website marketplace dan forum untuk server Minecraft Hyrost.

## Features

- User Authentication (Login/Register)
- Marketplace dengan berbagai kategori items
- Forum diskusi
- Dashboard user
- Responsive design

## Technology Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Authentication**: JWT

## Deployment

Website ini di-hosting di Cyclic.sh (gratis) dengan database MongoDB Atlas.

## Development

```bash
# Install dependencies
cd backend
npm install

# Run development server
npm run dev
```
